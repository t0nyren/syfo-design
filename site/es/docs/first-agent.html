Crear tu primer Agent

Siga el ejemplo del asistente de investigación para crear un Agent que trabaje de forma continua.

Esta guía le llevará 10 minutos para crear un Agent que realmente trabaje.

Agent que crearás

Un asistente de investigación (research-assistant): se encarga del trabajo de investigación en sus Channels —buscar información, hacer análisis de la competencia, entregar conclusiones en puntos clave—; además le añadirá un informe programado todos los días a las 9 AM para comprobar que puede trabajar de forma continua.

Paso 1: elegir dónde se ejecuta

Vaya a la página "Miembros" en la barra lateral izquierda y haga clic en "+ Crear Agent". El primer paso del asistente de creación es elegir dónde se ejecuta el Agent:

OpcionesAdecuado para
Syfo Cloud (recomendado)Funciona desde el primer momento, siempre en línea, disponible en cualquier momento desde el móvil y la web; elíjalo para Tasks programadas y Agents que deben estar en línea las 24 horas. Se factura según el consumo de créditos
My Own ComputerSe ejecuta en su propia computadora o servidor, con acceso directo a los archivos locales (por ejemplo, desarrollo y pruebas). No consume créditos; usted mismo configura el servicio de IA

El asistente de investigación necesita ejecutar un informe programado, así que seleccione Syfo Cloud y haga clic en "Siguiente".

Paso 2: configurar el Agent

El segundo paso es completar la información básica del Agent y elegir un modelo:

  • Nombre del Agent (obligatorio): research-assistant — su identificador único; solo admite letras, dígitos, guiones y guiones bajos
  • Nombre para mostrar (opcional): Asistente de investigación — se muestra en los mensajes y en la página de perfil; si se deja vacío, se usa el nombre del Agent
  • Descripción (opcional): busca información, realiza pequeñas investigaciones y entrega tarjetas de conclusiones

Luego elija el modelo y el nivel de razonamiento:

  • GPT-5.6 Terra: Capacidad equilibrada, adecuado para la mayoría de Tasks de ejecución, mejor relación costo-rendimiento
  • GPT-5.6 Sol: Modelo insignia, ideal para razonamiento complejo, investigación profunda y Tasks de cadena larga
  • GPT-5.6 Luna: Ligero y rápido, ideal para Tasks simples de alta frecuencia

El trabajo de investigación exige mucho razonamiento, así que elija GPT-5.6 Sol y mantenga el nivel de razonamiento en "Medio". Confirme y haga clic en "Crear Agent".

Después de la creación, el asistente de investigación aparecerá en el grupo de Agents de la lista de miembros; cuando su punto de estado se ponga verde, estará en línea.

Paso 3: invitar al Agent al canal

  1. Abra el Channel en el que quiera que colabore el Agent (por ejemplo, el Channel del proyecto #product-team; la captura de abajo usa #general en un espacio de demostración)
  2. Haga clic en el icono de miembros en la parte superior derecha del canal → Agregar miembros
  3. Busque el nombre del asistente y haga clic en el botón de agregar junto a la fila research-assistant (AGENT)

Paso 4: envíe el primer comando al Agent

Empiece con una tarea de investigación que pueda hacer de inmediato; @mencione al Agent directamente en el canal:

@research-assistant haz una investigación rápida sobre las estrategias de precios de nuestros tres principales competidores y publica aquí 3-5 puntos clave.

El Agent reclamará la tarea, comenzará a investigar y unos minutos después publicará sus conclusiones en el canal.

Luego asígnele una tarea programada para que siga trabajando:

@research-assistant a partir de ahora, todas las mañanas a las 9, resume la discusión del día anterior en este canal y publícala aquí con este formato:
1. Decisiones importantes, si las hay
2. Asuntos pendientes de seguimiento
3. Prioridades de hoy

Empieza mañana.

El Agent responderá con confirmación y configurará la tarea programada.

Paso 5: Verificar que el Agent esté listo

Pruébalo en DM:

@research-assistant ¿qué tareas programadas tienes ahora?

El Agent enumerará todos sus planes programados para confirmar que su configuración haya entrado en vigor.

¡Felicitaciones!

Su asistente de investigación está listo: el primer conjunto de puntos clave sobre la competencia ya se ha entregado, y mañana a las 9 AM enviará automáticamente el primer informe.
